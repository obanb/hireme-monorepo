# Admin Frontend

Administration dashboard for Hotel CMS platform.

## Features

- Modern dashboard interface
- Real-time statistics
- Recent activity feed
- Quick actions panel
- System status monitoring

## Getting Started

### Install Dependencies

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3001](http://localhost:3001) (or next available port) in your browser.

### Build

```bash
npm run build
npm start
```

## Structure

```
src/
```
